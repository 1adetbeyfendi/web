Download the native Cloud Player App to install it on your PC.

  • Create a shortcut to Cloud Player on your Desktop
  • Control the Cloud Player from your keyboard (play/pause, next track, previous track)
  • Have all the other nice features of Cloud Player

How to Install

Open the downloaded cloud-player.dmg file

After the Download has finished open the dmg file. Drag Cloud Player into the Applications folder

Start Cloud Player

We could not sign the Player with an official Apple developer certificate. When you try to start it, macOS will tell you that the application is from an unidentified developer.

To open the app anyway locate Cloud Player in the applications folder. Control click the Cloud Player app icon and the choose open from the shortcut menu.

This has to be done only when you start Cloud Player for the first time.

How to Install

Open the downloaded cloud-player.setup.exe file

After the Download has finished open the exe file.

We could not sign the Cloud Player with an official Windows developer certificate. Therefore the Windows Security Agent will prevent starting the installation process

To start the installation anyway click on the button "More Info" (1). A new button "Run anyway" will appear (2). Click on that button to start the installation process
